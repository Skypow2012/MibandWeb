# MibandWeb
control your Miband2 on website  
forked from [miband-js](https://github.com/vshymanskyy/miband-js)  

## DEMO
https://mw.imea.me

## FUNCTION
- sport data
- device info
- message notice
- phone notice
- vibrate notice
- shake mode
- heart rate monitor

## NOTICE
- Tested devices
    - Miband2
- Tested browsers
    - Chrome(Win)
    - Chrome(Android)
    
## RUN
```
npm install yarn -g
yarn
# development
yarn start
# production
yarn build
```
